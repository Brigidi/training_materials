# google sheets
google drive is a cloud based productivty ecosystem with a hierarchical identification number taxonomy and a native programming language platform, which makes it an optimal development environment (especially when leveraged within an enterprise account).

## when google sheets should be used
if your project needs to organize, visualize, and/or model data 

## where google sheets should be organized
google drive should be the primiary portal to your files. each of your files should be part of a project that is stored within a project folder

## why google sheets should be organized in project folders
organizing project components open up programming opportunities (aka helping computers) and makes it easier to share your work with others (aka helping people)

## how google sheets should be used
as of this writing, i have five types of tabs i recommend using:

1. red tabs, which are for data delivery
2. blue tabs, which are for data visualization
3. black tabs, which are for data configuration
4. orange tabs, which are for readme files
5. green tabs, which are holds for future data delivery (if needed)
