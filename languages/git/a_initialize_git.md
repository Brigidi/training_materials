the first step to using git version control is initializing git

## where you initialize git
open up the terminal window

## how you initialize git
make sure you have a `.gitignore` file in your repo ([here is a link to my standard file](/languages/git/recommended_gitignore.md)). the `.gitignore` will make sure you exclude files that you do not want commited to github (or your version control host of choice).

enter the following command:

```shell

git init

``` 

this will initialize git and allow you to start adding files 

```shell

git add -A

```

and making commits

```shell

commit -m "message_goes_here"

```
