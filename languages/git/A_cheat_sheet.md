this file aspires to introduce some commands that will help you start using git

| command | what it does |
| :---: | :---: |
| `git init` | initializes git within a project |
| `git branch` | tells you what branch you are in |
| `git -A` | adds all new files to the git repo |
| `git commit -m` | commits all added changes to the repo |
| `git commit -am` | commits all changes to files that have already been added to the repo. this will not work if the file was not added using `git -A` first|
| `git switch -c "new_branch_name"` | creates a new branch; replace `new_branch_name` with your branch name |
| `git push` | pushes all commits to the branch your currently working on |
| `git switch main` | returns to the main repo |
| `git pull` | pulls down repo from git source |
| `git branch -d new_branch_name` | deletes a branch; replace `new_branch_name` with your branch name |
