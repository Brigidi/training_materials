# delivering work vs. getting the job done
do you deliver work or do you get the job done

are you constantly fielding similar requests from the same ppl

## ways to handle requests
1. document them
2. push them to public slack channels
3. scale team knowledge so other ppl can help with them

## technical tools
1. documentation (who, what, when, where, why, and how)
2. database solutions
3. spreadsheet solutions
4. visulization solutions

## what to do when you get the same request repeatedly
- write down how to get the answer
- drive the request to a public forum
- tell your team about it and scale the effort

## what to do when you get requests from the same person
- meet with them
- identify their needs
- road map always-on solutions so they can serve themselves
- direct them to public channels, so the work can scale
