# move through your system
begin by opening a terminal window. if you are working in an Integrated Development Environment (ide) like PyCharm or VScode, then you probably have something called ```terminal```. it's also called ```terminal``` if you are using a mac computer running macOS.

i'm not sure what windows uses; i've seen things like ```powershell``` and ```windows terminal```.

**note**: *i use a mac, so the commands might vary for other systems; additionally, it might vary based upon what shell language is implemented (please reference the cheat sheet for more info on that).*

## open the terminal window
after opening the terminal window you should see a prompt like:

```shell

(base) your_username@your_computer_name ~ % 

```

this indicates that you are in your ```root directory```. the icon that follows your computer name — ```~``` — is called a tilde; it is the indicator for your root directory.

## figure out where you are
you can identify where you are by entering the ```pwd``` command, which stands for *present working directory*.

```shell

pwd

```
